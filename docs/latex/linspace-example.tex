\hypertarget{linspace-example}{}\doxysection{linspace}
Generate linearly spaced vector. (double x1, double x2, int n) generates n points. The spacing between the points is (x2-\/x1)/(n-\/1). \begin{DoxyReturn}{Returns}
pointer to the new vector
\end{DoxyReturn}
